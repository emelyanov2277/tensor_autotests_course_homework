Вот несколько заданий для работы с классом chels:

Задание на модификацию атрибутов:

Создайте нового персонажа с именем “Маша”, возрастом 25 лет, здоровьем 100, статусом True и хобби [“танцы”, “чтение”].

Измените возраст Маши на 26 лет и добавьте новое хобби “фотография”.

Выведите обновленную информацию о Маше.

Задание на работу с классом:

Напишите метод chel_health_heal, который будет восстанавливать здоровье персонажа на заданное значение.

Примените этот метод к персонажу Иван, восстановив ему 20 единиц здоровья.

Выведите список всех персонажей после лечения.

Задание на фильтрацию данных:

Добавьте метод get_adults, который возвращает список персонажей старше 18 лет.

Вызовите этот метод и выведите результат.

Задание на обработку ошибок:

Модифицируйте метод chel_health_damage так, чтобы здоровье не могло упасть ниже 0.

Попробуйте нанести урон 100 единиц персонажу Петя и убедитесь, что здоровье не уходит в минус.

Задание на сортировку:

Добавьте метод sort_by_health, который сортирует список персонажей по здоровью от большего к меньшему.

Выведите отсортированный список.

Задание на статистику:

Создайте метод average_health, который вычисляет среднее здоровье всех персонажей.

Выведите среднее значение здоровья.

Задание на удаление персонажа:

Добавьте метод remove_chel, который удаляет персонажа из списка по имени.

Удалите персонажа Петя и выведите обновленный список.
